# CBZ Viewer 操作説明

## GUI

### 初期操作

起動するとライブラリ画面が表示されます。

ライブラリ上部のパス欄に閲覧したいフォルダを指定してください。

指定方法:

* パス入力
* フォルダ選択ダイアログ
* フォルダのドラッグ＆ドロップ

指定したフォルダ内の本が一覧表示されます。
画像を直下に持つフォルダも本として表示されます。

本をダブルクリックするとビューアが起動します。

### 設定

設定画面は 一般 / ライブラリ / Viewer / パフォーマンス / 外部ツール の 5 タブです。

一般 > アプリ > 言語 で English / 日本語 を切り替えられます。

言語変更は即時反映され、再起動は不要です。

### ライブラリ設定

ライブラリ タブは 一覧 / カード表示 / 画像フォルダ に分かれています。

* 一覧 はカードサイズとホイール速度を調整します。
* カード表示 は HUD 文字とカード HUD / 選択カード表示の見た目を調整します。
* 画像フォルダ は画像フォルダを本として開くかどうかを切り替えます。

設定変更は即時反映されます。

### Viewer 設定

* 表示 は Viewer 全体の画質を調整します。
* 読書 は既定のページ開きと前回の読書位置から再開を切り替えます。
* 再構築後のアーカイブを別Viewerで開く は既定でOFFです。
  OFF では、現在のViewerは次の本へ移動するだけです。
  ON では、再構築後のアーカイブも別Viewerウィンドウで開きます。

### Performance

* L1 VRAM Cache: 先読み・履歴ページのGPU Textureを保持する容量
* L2 RAM Cache: デコード済みRGBAをRAMへ保持する容量
* Background Workers: バックグラウンド処理の並列数
* Danger Zone: 通常のハードウェア基準上限を超えて手入力する設定

## ライブラリ画面

### 基本操作

| 操作 | 内容 |
| --- | --- |
| ダブルクリック | 本を開く |
| Enter | 本を開く |
| Delete | 選択した本を削除 |
| F2 | 名前変更 |
| Ctrl+A | 全選択 |
| Ctrl+C | パスをコピー |

### 検索

検索ボックスから本名による絞り込みができます。

### お気に入り

お気に入り登録した本のみ表示できます。

### 読書状態

未読 / 読書中 / 既読で本を絞り込めます。件数も表示されます。

読書状態はビューアで読んだ結果から自動更新されます。途中で閉じると読書中、最後まで表示して閉じると既読になります。

### グループ

本を任意のグループへ分類できます。

設定方法は [本のグループ設定](book-groups.md) を参照してください。

### 右クリックメニュー

| 項目 | 内容 |
| --- | --- |
| 開く | 本を開く |
| Explorerで開く | エクスプローラーで表示 |
| お気に入りに追加/削除 | お気に入り切替 |
| グループを設定 | グループ分類 |
| 本固有設定をクリア | 本ごとの表示設定と読書状態を初期化 |
| 名前変更 | 名前変更 |
| コピー | パスをコピー |
| 削除 | 削除 |
| 外部ツール | 外部ツール実行 |

FolderBook では `フォルダへ移動` も表示されます。

選択した本体ファイルまたはフォルダは削除され、関連するサムネイル、Page Map、お気に入り、グループも削除されます。

複数選択時は一部の項目が表示されないか、無効になります。

## ビューア画面

### ページ移動

| 操作 | 内容 |
| --- | --- |
| ← / → / A / D | ページ移動 |
| PageDown | 次ページ |
| PageUp | 前ページ |
| Wheel Down | 次ページ |
| Wheel Up | 前ページ |
| Home | 先頭 |
| End | 最後 |

右開き / 左開きの設定により、← → と A D の進行方向は反転します。

### 本移動

| 操作 | 内容 |
| --- | --- |
| ↑ / ↓ | 本移動 |
| W / S | 本移動 |

ライブラリで表示されている本順序に従って前後へ移動します。

本と画像フォルダが混在していても同じ順序で移動できます。

### 表示

| 操作 | 内容 |
| --- | --- |
| F11 | フルスクリーン切替 |
| Space | スライドショー開始/停止 |
| ESC | 閉じる（フルスクリーン時は解除） |

### 本管理

| 操作 | 内容 |
| --- | --- |
| Delete | 現在の本を削除 |

画像フォルダを削除すると、フォルダ全体が削除されます。

### ページ範囲削除とアーカイブ再構築

この操作はアーカイブ本を再構築します。
Library から開いたアーカイブ本でのみ使用できます。
FolderBook、EPUB、ImageFile は対象外です。

すべての画像ページは削除できません。
失敗時は選択範囲を維持したまま再実行できます。

操作:

* `M`
  未選択なら、表示中ページ集合の小さいページを Start にします。
  Start のみなら、表示中ページ集合の大きいページを End にします。
  Start と End があるなら、表示中ページ集合の小さいページから再選択します。
* `Esc`
  範囲選択中は Clear Mark します。
  それ以外は既存動作のままです。
* `Delete`
  Start と End があるときは Delete Pages を開きます。
  それ以外は現在の本を削除します。
* 右クリック
  クリックした画像ページを Start / End に使います。
* 左下ヘルプ
  範囲選択中は `Range: S=...` / `Range: S=... E=...` を表示します。

### ツールバー

ツールバーから以下を操作できます。

* お気に入り
* 表示モード（AUTO / 単ページ / 見開き）
* ページ開き
* 表紙ブランク
* 画質
* スライドショー
* フルスクリーン
* 外部ツール

アニメーションWebPはアニメーションとして再生されます。見開き表示にも対応しています。

### ページ開き

右開き / 左開きでページ送りの向きが変わります。

設定の優先順位は次の通りです。

* 本別設定
* 未設定ならグローバル既定値

この設定は、左右キー、進捗バー、前後本カードの向きにも反映されます。

### 画質

4 モードがあります。

| 設定 | 意味 |
| --- | --- |
| 速度優先 | 生成速度を優先します |
| 標準 | 速度と画質の均衡を取ります |
| 画質優先 | 実表示サイズに対する高品質処理を優先します |
| 原寸優先 | 元画像解像度を安全上限内で維持します |

原寸優先では、読み込み時間と RAM / GPU メモリ使用量が増え、キャッシュ可能なページ数が減ります。

アニメーション画像には一部の画質処理が適用されません。

### 言語設定

English / 日本語 を切り替えられます。

既定は English です。

変更は即時反映されます。

## CUI（コマンドライン）

### 本を開く

```cmd
cbz-viewer.exe "C:\path\to\sample.cbz"
```

### 画像フォルダを開く

```cmd
cbz-viewer.exe "C:\path\to\books"
```

### 画像ファイルを開く

```cmd
cbz-viewer.exe "C:\path\to\page01.jpg"
```

対応画像ファイルから起動した場合は、親フォルダを画像本として開き、指定画像から表示を開始します。

### 複数起動

複数のビューアを同時に起動できます。

ウィンドウタイトルには現在開いている本のファイル名が表示されます。
